About this project

Pure frontend quiz platform built with HTML, CSS, and JavaScript. No backend—scores are stored using localStorage. Includes timer, progress, review, leaderboard, and certificate generation.

  • Press N / P for navigation during a quiz
  • Answers are auto-saved until you submit or restart
  • Pass mark is 60% for certificate
