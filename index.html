About this project

Pure frontend quiz platform built with HTML, CSS, and JavaScript. Scores stored using localStorage. Includes timer, progress, review, leaderboard, and certificate generation.

  • Keyboard shortcuts: N (next), P (previous)
  • Answers auto-saved until you submit
  • Pass mark: 60% for certificate
